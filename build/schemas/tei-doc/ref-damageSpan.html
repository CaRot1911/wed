Note

Both the beginning and ending of the damaged sequence must be marked: the beginning by the damageSpan element, the ending by the target of the spanTo attribute: if no other element available, the <anchor> element may be used for this purpose.

The damaged text must be at least partially legible, in order for the encoder to be able to transcribe it. If it is not legible at all, the damageSpan element should not be used. Rather, the gap or unclear element should be employed, with the value of the reason attribute giving the cause. See further sections 11.3.3.1. and 11.3.3.2. .
