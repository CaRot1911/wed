Class: UndoList

undo~ UndoList

An UndoList records operations that may be undone or redone. It maintains a single list of Undo objects the order by which they are passed to the record() method.

This object maintains a single history. So if operations A, B, C, D are recorded, C and D are undone and then E is recorded, the list of recorded operations will then be A, B, E.
