Methods

dismiss(obj)

Dismisses the popup. Calls the callback that was passed when the popup was created, if any.

Parameters:
Name Type Description
obj object | undefined

This should be the object selected by the user, if any. This will be passed to the dismiss_callback that was passed when the popup was created, if any. If you call this method directly and want a selection to occur, take care to use an object which is from the data set passed in the options parameter that was used when the popup was created. The value undefined means no object was selected.
