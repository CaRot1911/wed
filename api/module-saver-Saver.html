Class: Saver

saver~ Saver

A Saver is responsible for communicating with a server to save the data edited by a wed editor.

new Saver(url, headers, version, data_updater, data_tree)

Parameters:
Name Type Description
url String

The url location to POST save requests.
headers Object

Headers to set on the POST request. This may be necessary for cross domain request protection, for instance.
version String

The version of wed for which this object is created.
data_updater module:tree_updater~TreeUpdater

The updater that the editor created for its data tree.
data_tree Node

The editor's data tree.

Methods

recover(done)

This method is to be used by wed upon encountering a fatal error. It will attempt to record the last state of the data tree before wed dies.

Parameters:
Name Type Description
done function

A function to call once the recovery operation is done. Cannot be null or undefined. This function must accept one parameter which will be set to undefined if the method did not do anything because the Saver object is in an unintialized state or has already failed. It will be set to true if the recovery operation was successful, and false if not.
